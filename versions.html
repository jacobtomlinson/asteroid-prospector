v0.4

Changes in version 0.4:

  • Replaced the old ship design with a nice shiny new one with flame effects on the engines.
  • The timed countdown was changed to a fuel gauge which depletes over time, changing colour as it does.
  • The size of the space station was increased to make it more realistic when compared with the size of the ship.

v0.3

Changes in version 0.3:

  • Each asteroid provides a mixture of different resources, with different asteroid types giving different mixtures.
  • Addition of the space station
  • Pop-up hints, "Employee handbook" and the reset button added to the menu bar.
  • Added Waste as a resource type.

v0.2

Changes in version 0.2:

  • 3 different types of asteroid, each containing a different resource for collection.
  • A menu bar was added to the top of the window but was lacking any functionality.

The available resources were:

  • Fuels (Hydrogen & oxygen from ice)
  • Precious metals (Platinum)
  • Construction materials (Iron, Nickel)

v0.1

The first playable version of Asteroid Prospector had a single type of asteroid which could be destroyed and its waste collected for points.
